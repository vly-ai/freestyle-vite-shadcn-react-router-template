# Freestyle Vite + shadcn + React Router Template

Minimal static-site template for sandbox testing.

## Stack
- Vite + React + TypeScript
- React Router
- shadcn/ui primitives (Button)
- Tailwind CSS

## Quick start
```bash
pnpm install
pnpm dev
```

## Build
```bash
pnpm build
pnpm preview
```

## Notes
- Dev server binds to `0.0.0.0:3000` for sandbox preview compatibility.
- This repo is intended to be consumed as a git template/submodule source.
